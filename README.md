# 🌿 Đất Mũi Eco Farm - Vườn Mận Hồng ST Dr. Hùng

Chào mừng đến với kho lưu trữ mã nguồn của website **Đất Mũi Eco Farm**. Đây là website giới thiệu về mô hình canh tác mận hồng ST đặc sản tại vùng đất mũi Cà Mau.

## 🚀 Thông tin Website
- **Tên miền:** [datmuiecofarm.com](https://datmuiecofarm.com)
- **Nền tảng:** HTML5, CSS3, JavaScript
- **Triển khai (Deploy):** Netlify
- **Hệ thống quản lý bài viết (CMS):** Decap CMS (Netlify CMS)

## 📁 Cấu trúc thư mục quan trọng
- `/index.html`: Trang chủ chính của website.
- `/blog.html`: Trang danh sách các bài viết chia sẻ kiến thức.
- `/admin/`: Thư mục quản lý bài viết (Truy cập qua `/admin` trên trình duyệt).
- `/images/`: Chứa toàn bộ hình ảnh thực tế từ vườn mận.

## ✍️ Hướng dẫn đăng bài Blog
1. Truy cập `datmuiecofarm.com/admin`.
2. Đăng nhập bằng tài khoản Netlify Identity.
3. Tạo bài viết mới, thêm ảnh và nội dung, sau đó nhấn **Publish**.
4. Website sẽ tự động cập nhật sau khoảng 10-20 giây.
